1. Run index.html in the current directory (Mac OS or Windows OS)
2. Mouse movements:
	mouse wheel: 		zoom in/out
	hold left button: 	orbit and rotate map
	hold right button:	move map
	click left button: 	add a waypoint in a route
	click "Measure distance": start a segment between two waypoints for a route
	click right button: 	delete the last waypoint in a route

3.Labels:	
	Waypoint label contains: Latitude/Longitude & depth
	Segment  label contains: distance between waypoints & magnetic heading adjusted for local declination
